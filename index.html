08 Features

Every feature on this list exists because we've seen it move the needle on intro-offer claims and member retention. Nothing here is decoration.

Intro offer hero with one-tap claim

Above-the-fold intro offer ('First month $49,' '2 weeks unlimited $29,' '3 classes $39') with a one-tap claim flow. Tracked separately so you can A/B test pricing and creative.

INative class schedule, not a Mindbody iframe

Custom schedule UI that pulls live data from Mindbody, ClassPass, Zen Planner, Glofox, or Wodify via API where available. Branded, fast, filterable by class type and instructor.

Membership tier comparison built into the homepage

Transparent pricing — class pack, monthly unlimited, annual, family — with what's included at each level. Filters out price shoppers, qualifies serious prospects.s

Per-instructor profile pages

Each instructor gets a real page — bio, specialties, certifications (RYT-200, CrossFit-L1, CPR), training, class style, music preferences, photo gallery. Books fill faster when prospects can pick who teaches.

Facility tour video and class photo galleries

A 30–60 second studio walkthrough on the homepage. Per-class photo galleries showing real members, real instructors, real equipment. Fitness is sold on visible vibe.

Mobile-first booking with tap-to-call backup

Sticky bottom CTA on mobile with 'Book class' and 'Call studio.' Designed for the 75%+ of fitness traffic arriving on phones, mid-scroll on Instagram or Google.

Member transformation stories on every page

Real member photos and quotes — not stock — placed strategically on the homepage, intro offer page, and membership pages. Pre-and-post photos with permission, signed release forms included.

Virtual class library for studios with online offerings

For studios with on-demand or hybrid programs: a video library landing page with class previews, instructor bios, and signup. Converts at-home browsers who aren't ready for in-person yet.
